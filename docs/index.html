How it works

Two audio sources in, one clear cue out.

01 · CAPTURE

Both sides, natively

Your mic + the other party's system audio (ScreenCaptureKit). Who spoke is known by origin — no diarization.

02 · UNDERSTAND

On-device speech

SpeechAnalyzer transcribes live; Apple's Translation framework renders it in your language in ~100–200ms.

03 · COACH

What to say next

A local Claude Code CLI session reads the brief + your CV and streams a terse cue: guidance, a ready phrase, key words.

04 · GLANCE

Read in 2 seconds

A compact, always-on-top window beside your call. Emoji-cued, scannable under pressure.

Features

Built for the moment you're on the spot.

🎯

Friend-style coach

"He asked X → answer like this." Guidance in your language, a ready phrase in theirs, plus the vocabulary you need.

🌐

Live translation

Every line the other person says, translated instantly with the key words bolded for fast scanning.

📄

CV-grounded

Paste or import your résumé; the coach points at your real stories instead of making things up.

🎓

Voice training mode

A built-in interviewer reads your CV and grills you out loud, adapting to your answers — end-to-end practice.

🔒

Local-first & explicit

Speech and translation run on-device. Claude is keyless by default; DeepSeek is an explicit keyed option.

⌨️

Out of the way

⌥Space to show/hide, a menu-bar control, pin-on-top. It sits beside the call, not in front of it.

Training mode

Practice against a voice that reads your CV.

Turn on training mode and CueMe becomes its own interviewer: it speaks questions aloud from your brief and résumé, listens to your spoken answers, and asks natural follow-ups. It's a full end-to-end exercise of the real pipeline — no external app, no API key.

Install

Clone, open, run.

Requires macOS 26, Xcode 26, and the Claude Code CLI logged in for the keyless default brain. DeepSeek is optional.

# get it
git clone https://github.com/feliperun/cueme.git
open cueme/CueMe.xcodeproj # then ⌘R in Xcode

# the brain uses your existing Claude Code login
claude -p "hi" # should answer → you're set
